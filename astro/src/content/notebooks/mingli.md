### 1. 有没有命？

命之必然：人生遭遇的前定性
- **决定因素**：出生时间、出生地点、出身家庭。未经过本人同意，即被抛入这个世界，父母、环境、姓氏都是“既定之命”。
- **核心观点**：
    - 到四十岁还不信命，是“悟性太差”。
    - 命不是宿命论，而是对“人生无法选择起点”的一种哲学承认。
    - 孩子未来是否“成龙成凤”，多数由出生家庭和社会阶层决定，努力不是决定因素，教育主义者的理想往往落空。
    - 比如文革时候，“父亲叫走资派 母亲叫逃亡地主 我叫狗崽子“

命与教育的边界
- 命不能决定人格修养。教育最大价值是“教人做人”。
- 做一个“正直、有力量的人”，这是教育的目标，也是父母唯一可以深刻影响孩子的层面。
- 一旦认识到富贵穷通并不掌握在自己手中，就能将精力投入到“立身行道”“为世界贡献”中去。
- **“信命”不等于“等命”**，而是转向更积极的人生观，摆脱功利性的挣扎。
- 所以信命带来积极人生观，不信带来消极的人生观
- 假如把人生遭遇全都看成是主观的结果，那是唯心主义，唯意志论
- 但要是把所有人生遭际富贵穷通，都看成是大自然赋予你的阴阳五行结合的方式，这反而是唯物主义

2 - 命能不能算：
八字算命：
1. 唐山大地震死了24万人，这些人八字都不一样，有些人恐怕是大富大贵加长寿，怎么走了呢？
2. 文化大革命有多少做官的干部都离职被关到牛棚里去，难道他们都有这样的运吗？
3. 有多少孩子在同一个时辰出生，他们的命都一样吗？
其实就是总结为两点，国运规定，国运大于个人运，在稳定的社会八字更准。或者说有些人关到牛棚里自杀了，有的就活下来，然后官复原职混的更高。

所以国运能不能算？
黄极经世论 - 太乙神数，奇门遁甲，大六壬
八字 - 滴天髓，三命通会，穷通宝鉴，子平真诠
可以算，但是精确度不要过高，努力学也只能看到十之六七，已经不容易了
比如大概富贵程度，哪一行的，出生家庭地位，婚姻生活，有没有孩子
命理学没那么精确，看得更仔细，另有神通是无法学的
所以不要神秘化，就是朴素的，庸俗的学问
明朝的时候，命理学很普及，基本概念进入日常语言
家里只要有读书人都有三命通会
方言里“交了墓库运”，六冲，三合等，桃花也是命理学术语
所以命理学根据是什么：中国的宇宙观，阴阳五行之气的流变，中医也是这么认识人体的
疑惑很多：一个时辰内同样的八字，但恐怕实际的遭际是差异很大的
我的观点：同样的函数，不同的起点和环境也是不一样的，比如说山村里穷苦人家考进一本就是很厉害的，但是同样八字的人如果出生在大富大贵家，估计就是出国留学了。
两种情况：
1. 生辰八字一样，出生位置不一样，这个不能算，因为和方位有关，阴阳五行写出来其实分量权重不一样的，就可以区分开
2. 同一个地方就复杂了，他们必然有差别，但是基本的运势没差别。
3. 还有一种特殊情况：如果卡在两个时辰的分界线上，这两个人会有差别的，过一分钟气就转了，还得考虑真太阳时，夏令时，时区换算等
4. 还有剖腹产怎么办：找一个良辰吉时，说不用找，找了也没用，因为六亲，在你的和你妻子的八字里面已经看到了孩子，祖辈，父母，妻子等。就是说你得不得贵子已经明确了，你不可能说找个人算一下就改命，你这是造命啊。祖辈积的德一朝一夕就改了？一定要按照医学的要求，选好了个时间，结果没到，羊水破了，非剖不可啊。人算有什么用，不如天算啊。或者已经订好了，结果送进来一个抢救的，时辰耽误了。如果成功了，那是因为他八字本来就这个，配得上这个命格。尽人事听天命。

- **月令（月支/月建）**：月令对日主的旺衰影响重大，需重点分析。而月令虽然位于地支，但其本身却拥有对于整个命盘无可替代的掌控力和支配力，一旦得令，连天干都得看月支的眼色行事。
- **日柱的地支**（即**日支**）也具有重要意义。它被称为**配偶宫**，主要反映命主的婚姻状况、配偶特质以及与配偶的关系。此外，日支中的**藏干**（隐藏的天干）也会对命主的性格和命运产生影响。

|空亡位置|作用|实际含义|
|---|---|---|
|年支空亡|根基虚浮|与祖上疏离、童年环境不稳|
|月支空亡|运源断裂|青少年期漂泊、家庭缺位|
|日支空亡|几乎不出现|日支不会空亡（核心稳定）|
|时支空亡|子女/晚运弱化|子女不归、晚年孤寂|

- “得令”“得地”“得生”“得助”？
- 这里就不得不提到五行十天干与季节的关系了。就比如五行木，以甲木为例，只要出生在纯春季的月份里，严格来说才算是真正得令（因为春季有着木的最佳生长环境），生于其它的季节则都为失令。
- 再看春季的三个月份，分别为寅月、卯月和辰月。其中只有寅月和卯月的主气都为木，可以视作“纯春季”。而最后一个辰月由于位于春夏之交，杂气重，所以五行变为土，被排除在季令之外（其他三个季末月份“戌、丑、未”也为同样道理）
- 丙丁火为夏季得令（巳、午月）
- 庚辛金为秋季得令（申、酉月）
- 壬癸水为冬季得令（亥、子月）
- 戊己土为辰、戌、丑、未月得令
- 当然，结合十二长生我们也可以判断命主是否得令，即月支处长生、沐浴、冠带、临官、帝旺之地。
- 至于得令有啥好处，上面其实也已经有提到过。一旦得令则大概率日主身旺，一旦身旺就有了扛官杀和担财的资本，同时也能“令”命盘中的五行十神形成聚力。
- 而失令则恰恰相反，意味着失去了对命盘中五行十神的控制。天干地支的诸位不用再看月支的眼色行事，具体怎么样对待日主全靠自由发挥。
- **得地**，日干在其余各支中得长生（需阳日干），禄刃（地支中藏干的本气为比、劫）， 或逢墓库（需阳日干）
- **得生**，日干在命盘中遇正偏印
- **得助**，日干在命盘中遇比肩劫财（主要为天干透出）帮身

| 概念     | 定义       | 判断方式               | 意义  |
| ------ | -------- | ------------------ | --- |
| **得令** | 日主当令     | 月支为日主五行之旺月         | 根深  |
| **得地** | 日主藏干有根   | 四支中有生助比劫或禄、刃、长生、墓库 | 有根  |
| **得生** | 日主有印星生扶  | 有正印/偏印，力量适中        | 有母  |
| **得助** | 天干比肩劫财助力 | 有透出比劫来帮身           | 有兄  |
得令 > 得地 > 得生 > 得助  → 若四得皆无，则为“**日主失势**”，需用印比助命。

- 原理思想：之前我们讲五行十天干，可能更多地集中在它的形状上。比如甲木是参天大树、乙木是柔软藤蔓。但小白们也往往因此忽略掉它的形态。而它们毕竟不是一个个孤立封闭的剪影。五行当然也是有温度有生命的，它们也需要适宜的生长环境。所以它们对季节，以及季节带来的不同温度和湿度都是相当敏感的。同样被称为壬癸水，它有可能在夏日沸腾蒸发，也有可能在冬日冷冻结冰。日主以何种形态或状态出现在命盘中，自然会影响到命盘的整体结构与气场。所以是否得月令，才会成为人们断命的关键。
- 月柱掌管着一个人16-30岁的状态，是人生中最关键的阶段。
- 月令为运源，恰恰是连接“命与运”的纽带。月令就是大运的根源，大运的好坏直接取决于月令的好坏。

| 月令之力      | 表现         | 实质影响           |
| --------- | ---------- | -------------- |
| 控五行之旺衰    | 月支为主气，藏干最强 | 决定日主旺衰，间接决定用神  |
| 控“十神聚合方向” | 日主是否能担财、任官 | 身旺可任财官，身弱忌见    |
| 控“大运运行轨道” | 运是否助/损月令五行 | 运好坏基于月令为轴心     |
| 控人生16~30岁 | 月柱为青年期宫位   | 一生最关键十年，多由月令主导 |
**月令得地 = 结构稳定**；**月令遇冲 = 运势反复**

- **纯气地支**：
    - 地支中只有一种五行能量（即只藏一个天干）。
    - 例如：**卯**（乙木）、**酉**（辛金）。
    - 特点：力量集中，单一性强。
- **杂气地支**：
    - 地支中包含两种或三种不同的天干。
    - 例如：**辰**（戊土、乙木、癸水）、**巳**（丙火、庚金、戊土）。
    - 特点：能量复杂，多面性明显，依赖透干来展现作用。
- **透干**：当藏干中的某一天干透到天干时，其对应的五行和作用会被显现。例如：
    - 若八字中有“甲寅”，则寅木中藏的“甲木”直接透干，木的能量表现得特别明显。
- **合化冲动**：藏干可能因合局或冲克而激发。例如：
    - 若寅与亥形成合局（寅亥合木），则寅中的甲、丙可能同时被激发。
- **辰、戌、丑、未**这四个湿土或燥土地支被称为“墓库地支”，它们的藏干特别复杂，往往在大运、流年中展现独特影响。

| 类型      | 地支      | 藏干结构  | 特点          |
| ------- | ------- | ----- | ----------- |
| **纯气支** | 卯、酉、子、午 | 单一五行  | 力量集中，易断清晰   |
| **双气支** | 寅、申、巳、亥 | 主+副   | 次强，常带小变化    |
| **三气支** | 辰、丑、戌、未 | 主+中+余 | 势杂，变化多，易成大格 |
